# XSI Administrator Guide

## Dashboard Usage
- **Dashboard:** View high-level threat score, active alerts, and live event counts.
- **SIEM:** Search and filter security logs using advanced criteria (device, source, severity).
- **Endpoints:** Register new devices and manage existing ones. Click on a device to open its dedicated **Command Center**.
- **IPS:** Monitor and manage firewall/IPS rules.
- **Settings:** Configure system-wide parameters like retention and notifications.

## Device Command Center
- Provides an isolated view for each device, including:
  - Overview of security status.
  - Process monitor and network activity logs.
  - XDR and IPS event feeds specific to the device.
  - Ability to execute manual actions (e.g., quarantine file, isolate device).
